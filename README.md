# Pokedex
This is a program that will create a pokedex that cna be looked through and added to. 
The inputs are a file in the correct format(shown below), and user entries to navigate the menu.


A structure will be created within the porgram that can be used to search for any 'mon by their dex num, name, or type.
The user can choose to print the found pokemon(s) either in a seprete file, or in the temrinal
The outputs of the program will be the pokemon that was searched for, or multiple mons, printed in the terminal or on a file.
Have fun!

  
INPUT FILE REQUIRMENTS  (angled brackets do not need to be there)

{number of pokemon}
{pokemon index number} {name of pokemon} {pokemon type} {number of moves}
{move 1} {move 2} ... 
{pokemon index number} {name of pokemon} {pokemon type} {number of moves}
{move 1} {move 2} ...
  etc..
